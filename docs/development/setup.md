# Development Setup

## Prerequisites

| Tool | Version | Purpose |
|------|---------|---------|
| Node.js | 18+ | Frontend runtime |
| pnpm | 8+ | Package manager |
| Rust | 1.70+ | WASM compilation |
| wasm-pack | 0.12+ | WASM packaging |

## Quick Start

```bash
# Clone repository
git clone <repo-url>
cd dependency-cruiser-reporter

# Install dependencies (workspace)
pnpm install

# Build WASM module
cd packages/wasm
wasm-pack build --target bundler

# Start frontend development
cd ../frontend
pnpm dev

# Test CLI
cd ../cli
pnpm build
pnpm link --global
dep-report --help
```

## Package Development

### WASM Package (`packages/wasm`)

```bash
cd packages/wasm

# Build WASM module (for bundler)
wasm-pack build --target bundler

# Build for web (no bundler)
wasm-pack build --target web

# Run tests
wasm-pack test --headless --firefox

# Build release (optimized)
wasm-pack build --target bundler --release
```

### Frontend Package (`packages/frontend`)

```bash
cd packages/frontend

# Start dev server (http://localhost:5173)
pnpm dev

# Type check
pnpm typecheck

# Lint
pnpm lint

# Build for production
pnpm build

# Preview production build
pnpm preview

# Run E2E tests
pnpm test:e2e
```

### CLI Package (`packages/cli`)

```bash
cd packages/cli

# Build
pnpm build

# Link globally
pnpm link --global

# Run
dep-report --help
dep-report analyze --input input.json
dep-report open --file graph.json
```

## Workspace Commands

```bash
# Run from root directory

# Build all packages
pnpm -r build

# Run tests across all packages
pnpm -r test

# Lint all packages
pnpm -r lint
```

## Troubleshooting

### Node modules issues

```bash
# Clear cache and reinstall
rm -rf node_modules pnpm-lock.yaml
pnpm install
```

### WASM build errors

```bash
# Update toolchain
rustup update

# Add wasm target
rustup target add wasm32-unknown-unknown

# Rebuild
cd packages/wasm
wasm-pack build --target bundler
```

### Type errors

```bash
# Regenerate type declarations
pnpm typecheck
```

### CLI not found

```bash
# Re-link globally
cd packages/cli
pnpm link --global
```
